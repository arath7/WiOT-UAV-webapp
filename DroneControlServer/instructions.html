    UAV Control Instructions

    How to run: the Python scripts themselves are in the Scripts directory in the repository
  • Download the DroneControlServer Open in VSCode or other IDE, download PHP Server extension.
  • Open DroneControlServer/index.php and right click PHP Server: run project Have PHP server running.
  • Web application will be opened in web browser and from there, scripts will be run on a virtual or physical UAV
